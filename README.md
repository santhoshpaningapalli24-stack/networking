# Advanced Port Scanner 🔍

## Description
A multi-threaded GUI-based port scanner built using Python and Tkinter.
It supports scanning single IPs, IP ranges, and CIDR networks.

## Features
- Multi-threaded scanning
- Quick / Full / Custom port scan
- IP range & CIDR support
- Banner grabbing
- Export results (CSV & TXT)
- Real-time progress tracking

## Technologies Used
- Python
- Socket Programming
- Tkinter (GUI)
- Threading

## How to Run
1. Install Python
2. Run:
   python main.py
